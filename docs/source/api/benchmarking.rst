Benchmarking
============

Benchmarking provides the highest-level tabular synthesis workflow. Use
:class:`~synthyverse.benchmark.synthesis.TabularSynthesisBenchmark` when you want
repeatable train/validation/test splits, saved generator artifacts, synthetic
sampling, metric evaluation, and result CSVs from one configuration.
